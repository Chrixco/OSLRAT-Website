# Image Assets for OSLRAT

## Required Table Images

Please add the following images to this directory for the data carousel on the homepage:

### 1. table1-expected-damages.png
Screenshot or export of **Table 1** showing:
- Ranking of cities according to expected damages
- Under high-end and RCP8.5 sea-level rise scenarios
- Damages in billions of US dollars
- Cities ranked based on RCP8.5 in 2100

**Source:** Abadie et al., 2020, p. 7

### 2. table2-risk-assessment.png
Screenshot or export of **Table 2** showing:
- Ranking of cities according to risk (Expected Shortfall, ES(95%))
- Under high-end and RCP8.5 sea-level rise scenarios
- Damages in billions of US dollars
- Cities ranked based on RCP8.5 in 2100

**Source:** Abadie et al., 2020, p. 8

## Image Specifications

- **Format:** PNG (preferred) or JPG
- **Resolution:** High resolution for readability (recommended: 1200-2000px width)
- **Aspect Ratio:** Maintain original table aspect ratio
- **Quality:** Ensure text is clearly readable

## How to Add Images

1. Take a screenshot or export the tables from the source document
2. Save as PNG with the exact filenames above
3. Place in this directory (`assets/images/`)
4. The carousel will automatically display them

## Note

Until these images are added, the carousel will show placeholder elements. The website functionality will work once the images are placed in this directory.
